Dribdat Data Package
====================

This is an export of data about events, projects, participants and activities
generated with the Dribdat application for hackathons and other collaborative,
time-bounded events.

You can use this file to generate a static site using the backboard tool,
available at: https://codeberg.org/dribdat/backboard

Details about this file format are here: https://json.everyhack.day/
For general information about Data Package: https://frictionlessdata.io/

For more information and support of Dribdat, please visit https://dribdat.cc
